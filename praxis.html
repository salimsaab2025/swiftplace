Common billing gaps we've identified

1. Admin defaults to one code - path of least resistance

2. Don't know combination codes - arthroscopy + debridement + block = 3 codes, not 1

3. Fear of claim rejection - Conservative billing to avoid rejections

4. New codes not adopted - W8645 (ankle arthroscopy + reconstruction) added 2019, most don't know it

5. Never cross-reference insurers - same procedure, different fees per insurer

6. Fee schedules change - codes added/removed, fees updated regularly

How CCSD Codes Work With Insurers

All CCSD codes are universal - every UK private insurer (Bupa, AXA, Aviva, Vitality, WPA, Cigna, Healix, Freedom) uses the same code system. The code W2810 means the same thing to Bupa as it does to AXA.

What differs is the fee. Each insurer sets its own maximum reimbursement for each code. For example:

Bupa
W2810
£480
AXA
W2810
£450

£440

Fee-approved vs Fee-assured: If the surgeon is fee-assured with an insurer, they get 100% of the schedule fee. If only fee-approved, they may get 75-85%. This is the single biggest variable in actual reimbursement.
